# Student Management System

A console-based Student Management System developed in Java. This project allows users to manage student records using a simple menu-driven interface. It demonstrates Java fundamentals such as Object-Oriented Programming (OOP), Collections, loops, conditional statements, and user input handling.

## Features

- Add a new student
- View all student records
- Search a student by ID
- Update student details
  - Name
  - ID
  - Attendance
- Remove a student
- Exit the application

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework (`ArrayList`)


## Project Structure

```
StudentMgmt.java
 ├── StudentMgmt (Main Class)
 └── Student (Model Class)
```

## How to Run

1. Clone this repository.
2. Open the project in Eclipse, IntelliJ IDEA, or VS Code.
3. Compile and run `StudentMgmt.java`.
4. Follow the on-screen menu to perform operations.
